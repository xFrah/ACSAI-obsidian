Computing Standard Deviation

Standard deviation is calculated as follows:

  1. Calculate the mean of all data points.
  2. Calculate the Variance for each data point.
  3. Square the variance of each data point (from Step 2).
  4. Sum of squared variance values (from Step 3).
  5. Divide the sum of squared variance values (from Step 4) by the number of data points in the data set less 1.
  6. Take the square root of the quotient (from Step 5).
Formula explanation

We are taking the average of the distances from the mean for each point.
But, we square the distances so that they are always positive, then we sqrt everything to get the correct unit of measurament.
